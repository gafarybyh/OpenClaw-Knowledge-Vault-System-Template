# TOOLS: Raynor Shortcuts

- `message`: Send/manage telegram messages with attachments (files, images, etc.) to targets.
- `graph-search`: Traverse Knowledge Graph via run `node /vault/scripts/graph/graph-search.mjs <query>`.
- `memory_search`: Semantic search in memory and past sessions.
- `sessions_spawn`: Offload heavy work to specialized workers (Context GC).
- `rtk <cmd>`: **RTK prefix** command wrapper for exec/shell to minimize tokens.
- `web_search`: Real-time web research (Tavily/Web Search).
- `browser`: Web automation and multi-step browser flows.
- `gog`: Google Workspace CLI (Gmail, Drive, Docs, etc.).
- `pls-office-docs`: Generate/manipulate Office docs (PDF, DOCX, XLSX, PPTX).
- `diagram-maker`: Create SVG or Excalidraw diagrams.
- `clawhub`: Manage agent skills via ClawHub CLI.
- `mcporter`: Manage and inspect MCP servers.
- `skill-creator`: Create, edit, and audit AgentSkills.
- `taskflow`: Coordinate multi-step detached tasks.
